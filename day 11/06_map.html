[map] key : value 형태의 데이터를 한개의 배열로 박들어서 다시 배열에 저장하는 방식

  • new Map() new Map([[key,value],[key,value],[key,value]]);
  • map 은 기본적으로 중첩구조를 가지고 있음, key 중복 불가능.
  • key 를 식별자로 하는 get(keyName), set(keyName, value) 메서드 제공
  • key 는 문자로 하는 것을 원칙
  • delete() / has() / forEach() / keys() / value() / entries() / size() / clear()
