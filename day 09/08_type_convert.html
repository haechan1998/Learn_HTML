타입 변환 casting (형변환)

  • 문자 -> 숫자 : Number('String'), parseInt('1'), parsfloat('0.5')
  • 숫자 -> 문자 : 123.toString();
  • 문자 -> boolean : Boolean('true');
  • boolean -> 문자 : String(true);
  • number -> boolean : Boolean(1) => true / Boolean(0) => false
  • boolean -> number : Number(true) / Number(false)
