Event : html 객체에 줄 수 있는 전역객체로서 모든 태그 객체에 참조 시킬 수 있다.

  • a tag / button tag 에 이벤트를 주로 참조시켜 사용 클릭에 대한 방응을 기대 할 수 있게 만드는 것이 일반적
  • a tag 는 href 이동속성을 막는 코드가 필요
  • 객체에 이벤트를 부여하는 방법은 크게 3가지로 구분
  • 각자 목적과 작동방식이 다름
  • 화면의 출력영역을 Document영역(객체) 라고 한다.
  • DOM 영역
  • tag = html element, 속성 = attribute, 일반텍스트 = textNode 라고 함.
  • html로 출력하기 : innerHTML 라는 property 를 사용
  • text 로 출력하기 : innerText 라는 property 를 사용
  • html element 를 가져올 때 : getElementById / getElementByclassName
  • html element 를 가져올 때 : querySelector('selector') => # .
