document 영역에서 사용되는 이벤트를 정의

  • 마우스 이벤트 : click(), focus(), input(), change(), mouseover(), mouseout(), scroll()...
  • 키보드 이벤트 : keyDown(), keyPress(), KeyUp()
  • 기타 이벤트 : preventDefault() // a태그의 이벤트를 막을 때 사용한다. (a 태그와 event 가 같이 있을 경우.) target {submit(), reset()} => form 태그 안에서 사용.
  • addEventListener('eventName', callback function); 메서드를 사용하여 구현하는 것을 권장
  • 하나의 객체에 이벤트 중복 부여 가능
