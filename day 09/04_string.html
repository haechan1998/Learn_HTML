String : 가장 많이 다루는 데이터 타입

  • java 에서의 String method 와 거의 유사함.
  • 문자 데이터는 index(0 부터 시작)를 가지고 있음.
  • 문자열 값 내에서 <br> 개행을 하는 것은 추천하지 않음.
  • length : 문자열의 총 길이
  • new String() : 객체 타입으로 선언 가능 / 잘 쓰이지 않음
  • 추출 관련 메서드 : substring(), substr(), slice()
  • 문자 대체 관련 메서드 : replace(str, old, new) = 처음 찾은 값만 가능.
  • trim() : 양쪽 공백 제거 // 화면에서 우선 처리
  • padStart(str, 길이, 채울문자) / padEnd() 010104-1******
  • charAt(index) / indexOf(str, 문자)
  • startWith(str, 문자) / endWiht() 해당 문자가 str 의 앞 / 뒤에 존재하는지 확인하는 메서드 => return true / false
